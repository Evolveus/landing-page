# Evalify Feature Overview

Evalify is a complete digital assessment platform for educational institutions. It helps institutions plan courses, manage users, create question banks, conduct secure online exams, evaluate submissions, and review student performance from one centralized system.

The platform is designed for day-to-day academic use by administrators, semester managers, faculty members, and students. It supports both regular classroom assessments and controlled lab-based examinations.

## Key Benefits for Institutions

- Centralized management of departments, semesters, batches, courses, users, labs, question banks, quizzes, submissions, and results.
- Role-based access for administrators, managers, faculty, and students so each user sees only the tools relevant to them.
- Structured academic setup that matches institutional workflows: departments, batches, semesters, courses, instructors, students, and labs.
- Secure quiz delivery with password protection, fullscreen mode, kiosk mode, lab/IP-based restrictions, and violation tracking.
- Reusable question banks that reduce repeated faculty effort and improve consistency across assessments.
- Support for multiple question formats, including objective, descriptive, file upload, matching, fill-in-the-blank, and coding questions.
- Result dashboards and detailed evaluation views for faster academic review.
- Student-friendly dashboards showing active, upcoming, completed, and missed quizzes.
- Scalable deployment support for institution-owned infrastructure.

## Institution Administration

Evalify gives the institution a dedicated administrative layer to manage the academic structure.

### User Management

- Create, update, search, and manage users.
- Support for multiple roles: company, admin, manager, faculty, and student.
- Maintain user status and profile information.
- Assign students to batches.
- View and manage user records from a central dashboard.
- Support for profile image upload and management.

### Department Management

- Create and manage academic departments.
- Activate or deactivate departments when needed.
- Use departments as the base for organizing batches and students.

### Semester Management

- Create and manage academic semesters.
- Assign semester managers who can oversee courses and assessments within their semester.
- Manage active and inactive semesters.
- Bulk-create semesters where needed.

### Batch Management

- Create and manage student batches.
- Track joining year, graduation year, section, department, and active status.
- Add or remove students from batches.
- View available students for batch assignment.

### Course Management

- Create and manage courses with course name, code, description, type, semester, and status.
- Support course types such as core, elective, and micro-credential courses.
- Assign instructors to courses.
- Assign students directly to courses.
- Assign batches to courses.
- Bulk-create courses using spreadsheet-based workflows.
- Check duplicate course data before creation.

### Lab Management

- Create and manage institution labs.
- Store lab block/location details.
- Configure lab network restrictions using IP subnets.
- Use labs to restrict quizzes to approved physical locations.

## Faculty and Manager Features

Faculty members and semester managers get tools to manage assessments efficiently.

### Faculty Dashboard

- View assigned courses.
- Track active, upcoming, and completed quizzes.
- See question bank count and student reach.
- Access recent quizzes quickly.
- Navigate directly to courses, quiz creation, quiz results, and question banks.

### Course-Level Assessment Management

- View courses assigned as an instructor or manager.
- Search and filter courses.
- View students connected to assigned courses.
- Select courses while creating assessments.
- Assign assessments to courses, batches, selected students, and labs.

## Question Bank Management

Evalify includes reusable question banks so faculty can build and maintain high-quality assessment material over time.

### Question Banks

- Create, update, and delete question banks.
- Organize banks by name, course code, and semester.
- Search, sort, and filter question banks.
- View question count, topic count, and sharing information.
- Share banks with other users.
- Control shared access levels.
- Remove sharing access when required.

### Topic Organization

- Create topics inside question banks.
- Organize questions topic-wise.
- Update or delete topics.
- Filter and list questions by topic.

### Supported Question Types

Evalify supports a wide range of assessment formats:

- Single-correct multiple choice questions.
- Multiple-correct multiple choice questions.
- True or false questions.
- Descriptive questions.
- Fill-in-the-blank questions.
- Match-the-following questions.
- File upload questions.
- Coding questions.

### Academic Tagging and Quality Controls

Questions can include academic metadata that helps institutions align assessments with curriculum goals:

- Difficulty level: easy, medium, or hard.
- Course outcome mapping from CO1 to CO8.
- Bloom's taxonomy level: remember, understand, apply, analyze, evaluate, and create.
- Marks and negative marks.
- Explanation or solution notes.
- Topic association.

### Bulk Question Upload

- Upload MCQ questions using spreadsheet templates.
- Validate uploaded question rows before saving.
- Include marks, negative marks, difficulty, Bloom's taxonomy, course outcome, explanation, options, and correct answers.
- Download a ready-to-use MCQ template.

## Quiz and Exam Creation

Evalify supports the full quiz creation workflow from planning to publication.

### Quiz Details

- Add quiz title, description, and instructions.
- Set quiz duration.
- Configure start and end date/time.
- Add tags for easier organization.
- Track question breakdown by difficulty and total marks.

### Participant Selection

- Assign quizzes to one or more courses.
- Assign quizzes to batches.
- Assign quizzes to individual students where required.
- Restrict quizzes to selected labs.
- Search students, batches, and labs while assigning participants.
- Automatically align batch selections with selected courses.

### Quiz Structure

- Create sections within quizzes.
- Add questions directly to quizzes.
- Add questions from existing question banks.
- Reorder sections.
- Reorder questions.
- Move questions between sections.
- View all questions included in a quiz.

### Quiz Controls

Institutions can configure exams based on the level of control required:

- Publish or unpublish quizzes.
- Publish or hide results.
- Password-protect quizzes.
- Enable auto-submit at the end of the exam.
- Enable or disable calculator access.
- Enable fullscreen mode.
- Enable kiosk mode.
- Shuffle questions.
- Shuffle answer options.
- Use linear quiz flow where students proceed in order.

## Secure Exam Delivery

Evalify includes multiple controls to help institutions conduct controlled online examinations.

### Fullscreen Enforcement

- Require students to remain in fullscreen mode during exams.
- Record fullscreen exits as violations.
- Show a blocking prompt asking the student to return to fullscreen.

### Violation Tracking

The platform can track and record exam violations, including:

- Tab switching.
- Window focus loss.
- Fullscreen exit.
- Suspicious window resizing.
- Copy, paste, and cut attempts.
- Right-click/context-menu attempts.
- Developer tool shortcuts.
- Print/save shortcuts.
- Screenshot shortcut attempts.
- Restricted keyboard shortcuts.

### Kiosk Mode

- Restrict exams to an approved kiosk environment.
- Block access if the required kiosk validation is missing.
- Useful for institution-managed exam browsers or controlled lab environments.

### Lab and Network Restriction

- Restrict quizzes to selected labs.
- Validate student access using configured lab IP subnets.
- Helps ensure certain exams can only be attempted from approved campus locations.

## Student Exam Experience

Students get a focused exam interface designed for clarity and reliability.

### Student Dashboard

- View active courses.
- See live quizzes.
- See upcoming quizzes.
- Track completed and missed quizzes.
- Navigate to all courses and quizzes.

### Quiz Access

- View quizzes assigned through courses, batches, or direct assignment.
- See quiz status such as active, upcoming, completed, or missed.
- Open quiz instructions before starting.
- Start only eligible and published quizzes.

### Exam Interface

- Timer-based exam experience.
- Question navigation panel.
- Section-aware question display.
- Answer saving during the exam.
- Auto-submit support.
- Manual submission support.
- Persistent exam state for smoother recovery during an active exam.

### Student Answer Support

Students can answer different question types in suitable formats:

- Select options for MCQ and multiple-correct questions.
- Enter text for descriptive and fill-in-the-blank questions.
- Match related items.
- Upload files where permitted.
- Write code for coding questions.

## Coding Assessment Features

Evalify supports programming-based assessments for technical courses.

- Coding questions with configurable programming languages.
- Support for languages such as Java, Python, C++, JavaScript, C, Octave, and Scala.
- Boilerplate code and driver code support.
- Reference solutions for evaluation setup.
- Visible and hidden test cases.

## Evaluation and Results

Evalify helps faculty review outcomes at both student and question level.

### Evaluation Settings

- Configure MCQ partial marking.
- Configure fixed or percentage-based negative marking.
- Configure coding partial marking.
- Store evaluation settings per quiz.

### Result Management

- View quiz result statistics.
- View student-wise responses.
- View question-wise responses.
- Open detailed student result pages.
- Review individual question performance.
- Update evaluation results manually where required.
- Track submission status and evaluation status.

### Submission Tracking

- Track whether a student has not submitted, submitted, or was auto-submitted.
- Store quiz start time, end time, submission time, duration, score, and total score.
- Record IP information and exam violations.
- Identify submissions with violations.

## Security and Access Control

Evalify is built around controlled access and institutional data separation.

- Role-based access control.
- Protected routes for authenticated users.
- Separate permissions for admins, managers, faculty, and students.
- Tenant-aware data structure for institution-level separation.
- Row-level security support at the database layer.
- Secure file upload workflows.
- Password-protected quiz access.
- Kiosk and lab restrictions for high-stakes exams.

## Deployment and Operations Readiness

Evalify is prepared for institutional deployment and operations.

- Web-based access for administrators, faculty, and students.
- Can be deployed on institution-managed infrastructure.
- Supports production-ready application setup.
- Supports organized database setup and maintenance workflows.
- Includes background processing for time-based exam actions such as auto-submit.
- Supports reliable exam-state handling during active exams.
- Supports secure storage of files and images.
- Includes analytics support for understanding platform usage.

## Summary

Evalify gives institutions a complete assessment management system, from academic setup to secure exam delivery and result analysis. It reduces manual work for administrators and faculty, improves exam control, supports diverse question formats, and gives students a clear, organized assessment experience.
